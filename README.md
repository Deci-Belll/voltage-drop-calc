# 船舶用電圧降下計算ツール (Voltage Drop Calculator for vessel)

船舶の電気設備の設計において重要な、電線の電圧降下および電圧降下率を計算し、適切な電線断面積（sq）を選定するためのデスクトップアプリケーションです。

## 主な機能

- **電圧降下計算**: 三相交流、単相交流、および直流に対応した計算ロジック。
- **電線自動選定**: 許容される電圧降下率に基づき、最適な電線サイズ（SQR）を自動的に提案します。
- **グラフ表示**: 電線断面積の変化に伴う電圧降下率の推移を可視化。
- **GUI入力**: Tkinterを使用した直感的な操作パネル。

## 必要要件

- Python 3.8以上
- Matplotlib
- Tkinter (通常、Python標準ライブラリに含まれています)

## インストールと実行

1. リポジトリをクローンまたはダウンロードします。
2. 必要なライブラリをインストールします。
   ```bash
   pip install matplotlib
   ```
3. アプリケーションを起動します。
   ```bash
   python main.py
   ```

## 使い方

1. **電源種別**を「三相 AC」「単相 AC」「直流」から選択します。
2. **電圧、電流、電線長、本数、現在の断面積**を入力します。
3. **許容電圧降下率**を設定します。
4. 「計算開始」ボタンをクリックすると、結果とグラフが更新されます。

## ファイル構成

- `main.py`: Tkinterを使用したGUIの構築およびイベント制御。
- `calculations.py`: 計算ロジック、定数（SQR, 抵抗値）、およびグラフ作成処理。

## 計算式について

本ツールでは、電線の温度上昇等を考慮した係数（1.3）を含む実用的な簡易計算式を使用しています。

本ツールは舶用電線を対象とした計算を行っています。陸上向けですと計算方法が異なる可能性がありますので注意してください。
